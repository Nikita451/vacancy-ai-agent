# AI Career Path Analyzer 🚀

Мультиагентная система на базе **LangGraph** и **PydanticAI**, которая превращает название вакансии в детализированный план развития с анализом рынка и зарплат.

## 🌟 Особенности
- **Автономный поиск**: Использует **Tavily API** для анализа реальных вакансий и статей 2025–2026 годов.
- **Цепочка агентов**:
    - `Scout`: Находит актуальный стек технологий и тренды.
    - `Finance`: Оценивает вилки зарплат (Москва, Регионы, Remote).
    - `Mentor`: Формирует дорожную карту обучения и подбирает ресурсы.
    - `Auditor`: Проверяет отчет на логическую целостность (Self-Correction).
- **Строгая валидация**: Все данные типизированы через **Pydantic v2**.
- **Современный стек**: Управление зависимостями через `uv`, оркестрация через `LangGraph`.

## 🛠 Технологии
- **Язык**: Python 3.13+
- **Frameworks**: `pydantic-ai`, `langgraph`
- **LLM**: GPT-4o-mini (via OpenRouter)
- **Search API**: Tavily
- **Environment**: `uv`

## 🚀 Быстрый старт

### 1. Подготовка окружения
Убедитесь, что у вас установлен [uv](https://github.com).

```bash
# Клонировать репозиторий
git clone https://github.com/Nikita451/vacancy-ai-agent
cd career-path-analyzer

# Создать виртуальное окружение и установить зависимости
uv sync
```

### 2. Настройка секретов
Создайте файл `.env` в корне проекта на основе шаблона:
```bash
cp .env.example .env
```
Добавьте ваши ключи:
- `OPENROUTER_API_KEY`: Ваш ключ от OpenRouter.
- `TAVILY_API_KEY`: Ваш ключ от Tavily.

### 3. Запуск анализа
Система сгенерирует отчеты для предустановленных кейсов (или измените список в `main.py`):
```bash
uv run python main.py
```

## 📊 Примеры отчетов
Готовые JSON-отчеты с логами работы (поле `logs`) и метриками телеметрии доступны в папке `examples/`. Они содержат:
- `generated_at`: Точное время генерации.
- `quality_score`: Вердикт агента-аудитора.
- `learning_path`: Пошаговый план обучения на 4 фазы.
